# Roadmap

## Integração com Análise de Perfil

### MVP (prioritário)
- Suportar upload de PDF exportado do LinkedIn
- Processar o PDF e guardar resultados no backoffice
- Exibir resumo do score e recomendações principais
- FAQs por passo para apoiar utilizadores inexperientes
- Multi-lingue (PT-PT, EN, ES, FR, DE)
- Chat gratuito ligado ao utilizador
- Histórico de atividade (análises, roadmaps, insights)

### Extra (fase seguinte)
- Suportar link do perfil (avaliar viabilidade técnica e termos)
- Atualização periódica do perfil a partir do link
- Integração via API (fase futura de desenvolvimento)
- Histórico de análises e comparativos
- Exportação de relatórios (PDF/Excel)
- Pagamentos (Stripe) e plano freemium/premium
- NIF obrigatório apenas no momento de pagamento
- Chat com base de conhecimento + histórico
- Geração de conteúdo (premium)
- Monitorização de mudanças do LinkedIn (alertas e revisão)
- Atualizações periódicas das regras 360Brew com base na documentação
